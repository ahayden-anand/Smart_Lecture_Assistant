A lightweight, accessibility-focused web application that assists lectures through **real-time speech recognition**, **live transcription**, **automated summaries**, and **voice-based interaction**.

---

## Project Overview

**Smart Lecture Assistant** is an interactive frontend application developed for **GK Hacks 2025**. The project demonstrates how modern web technologies can be combined with speech services to improve lecture accessibility, engagement, and information retention.

Built using **TypeScript** and **Vite**, the application integrates on-device or cloud-based speech capabilities to deliver live lecture assistance in real time.

---

## Key Features

###  Speech & Transcription
- Real-time speech recognition with continuous and single-utterance capture  
- Live transcript display with dynamic topic tracking  

###  Intelligence & Assistance
- Automated summaries of lecture segments  
- Voice-driven Q&A panel to ask contextual follow-up questions  

###  Audio & Interaction
- Text-to-speech playback for generated responses  
- Audio waveform visualization for live feedback  

###  Export & Accessibility
- Export transcripts and session data  
- Accessibility panel with adjustable presentation settings and controls  

---

##  Tech Stack

- **TypeScript** – Type-safe frontend development  
- **Vite** – Fast build and development tooling  
- **Web Speech APIs / Cloud Speech Services** *(configurable)*  
- **Modern Web APIs** for audio, visualization, and accessibility  

---

##  Use Cases

- Lecture assistance for students  
- Accessibility support for learners with disabilities  
- Live note-taking and content summarization  
- Educational technology demonstrations
